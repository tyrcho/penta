**Pourquoi cette correspondance ?**

- **Magic** : Les couleurs ont des **relations d'opposition** (ex: Blanc ↔ Noir, Bleu ↔ Rouge) et des **synergies** (ex: Vert et Blanc pour l'harmonie).
- **Cycle adapté** : Chaque élément **engendre** le suivant (Bois → Feu → Ombre → Cristal → Lumière → Bois) et **contrôle** un autre.
- **Fusion** : On combine les deux pour créer un système **cohérent et riche** :
    - **G (Bois) → R (Feu) → B (Ombre) → U (Cristal) → W (Lumière) → G (Bois)** _(Cycle Engendre)_.
    - **G (Bois) ⚔️ B (Ombre)** / **R (Feu) ⚔️ U (Cristal)** / **B (Ombre) ⚔️ W (Lumière)** / **U (Cristal) ⚔️ G (Bois)** / **W (Lumière) ⚔️ R (Feu)** _(Cycle Contrôle)_.

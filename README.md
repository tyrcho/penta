**Pourquoi cette correspondance ?**

- **Magic** : Les couleurs ont des **relations d'opposition** (ex: Blanc ↔ Noir, Bleu ↔ Rouge) et des **synergies** (ex: Vert et Blanc pour l'harmonie).
- **Cycle adapté** : Chaque **Thème** **engendre** le suivant (Nature → Chaos → Ombre → Science → Loi → Nature) et **contrôle** un autre.
- **Fusion** : On combine les deux pour créer un système **cohérent et riche** :
    - **Nature (G) → Chaos (R) → Ombre (B) → Science (U) → Loi (W) → Nature (G)** *(Cycle Engendre - par Thème)*.
    - **Bois (G) → Feu (R) → Mort (B) → Cristal (U) → Lumière (W) → Bois (G)** *(Cycle Engendre - par Ressource)*.
    - **G ⚔️ B** / **R ⚔️ U** / **B ⚔️ W** / **U ⚔️ G** / **W ⚔️ R** *(Cycle Contrôle - par couleur)*.

---

**Légende** :
- **Thème** = Concept narratif (Nature, Chaos, Ombre, Science, Loi)
- **Ressource** = Ressource utilisée dans le jeu (Bois, Feu, Mort, Cristal, Lumière)
